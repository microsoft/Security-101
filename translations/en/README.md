# üöÄ Cybersecurity for Beginners ‚Äì a curriculum

In this fast-paced era of AI technology adoption, understanding how to secure IT systems is more important than ever. This course is designed to teach you the basics of cybersecurity to help you start your learning journey. It is vendor-neutral and divided into short lessons that take about 30-60 minutes to complete. Each lesson includes a short quiz and links to additional resources for deeper exploration.

![Cybersecurity for Beginners](../../translated_images/banner.cc5b05d7e5deed065123ba68678b48cbbfe411cb264c09cec64f58eda064a28a.en.jpg)

What this course covers üìö

- üîê Basic cybersecurity concepts like the CIA triad, and the differences between risks, threats, etc.
- üõ°Ô∏è Understanding what a security control is and the different forms it can take.
- üåê Understanding zero trust and its importance in modern cybersecurity.
- üîë Key concepts and themes across identity, networking, security operations, infrastructure, and data security.
- üîß Examples of tools used to implement security controls.

What this course does not cover üôÖ‚Äç‚ôÇÔ∏è

- üö´ How to use specific security tools.
- üö´ How to "hack" or perform red teaming/offensive security.
- üö´ Information about specific compliance standards.

[![Watch the video](../../translated_images/intro_placeholder.f42382df518f233a1ea3cb1c82ae8f92732bc3ac4ac2b3138cb561d24ca91df5.en.png)](https://learn-video.azurefd.net/vod/player?id=a0fe1cef-c064-4d59-97a9-e89e12a99b4d)

Once you‚Äôve completed this course, you can continue your learning with some of our Microsoft Learn modules. We recommend starting with [Microsoft Security, Compliance, and Identity Fundamentals.](https://learn.microsoft.com/training/paths/describe-concepts-of-security-compliance-identity/?WT.mc_id=academic-96948-sayoung)
