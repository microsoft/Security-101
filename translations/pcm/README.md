# üöÄ Cybersecurity for Beginners ‚Äì na curriculum

For dis time wey AI dey grow fast, e dey very important to sabi how to protect IT systems. Dis course go teach you di basic cyber security ideas wey you need to start your security learning. E no dey focus on any vendor and e dey divide into small lessons wey go take like 30-60 mins to finish. Each lesson get small quiz and links wey go help you read more if you wan sabi di topic well well.

![Cybersecurity for Beginners](../../translated_images/banner.cc5b05d7e5deed065123ba68678b48cbbfe411cb264c09cec64f58eda064a28a.pcm.jpg)

Wetin dis course go teach üìö

- üîê Basic cyber security ideas like di CIA triad, di difference between risks, threats, etc.
- üõ°Ô∏è Wetin security control mean and di different types wey e get.
- üåê Wetin zero trust mean and why e dey important for modern cyber security.
- üîë Key ideas and themes for identity, networking, security operations, infrastructure and data security.
- üîß Examples of tools wey dem dey use to implement security controls.

Wetin dis course no go teach üôÖ‚Äç‚ôÇÔ∏è

- üö´ How to use specific security tools.
- üö´ How to "hack" or do red teaming/offensive security.
- üö´ Learning about specific compliance standards.
