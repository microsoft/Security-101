# üöÄ Keamanan Siber untuk Pemula ‚Äì sebuah kurikulum

Di era adopsi teknologi AI yang berubah dengan cepat ini, memahami cara mengamankan sistem TI menjadi semakin penting. Kursus ini dirancang untuk mengajarkan konsep dasar keamanan siber guna memulai pembelajaran keamanan Anda. Kursus ini tidak bergantung pada vendor tertentu dan dibagi menjadi pelajaran-pelajaran kecil yang dapat diselesaikan dalam waktu sekitar 30-60 menit. Setiap pelajaran memiliki kuis kecil dan tautan untuk bacaan lebih lanjut jika Anda ingin mendalami topik tersebut.
